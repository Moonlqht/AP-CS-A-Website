Arrays

If you were to store the information about all the students attending a school, how would you do it? It obviously seems impractical to create thousands of variables, one for each student. This is where arrays are useful! Arrays act as references to consecutive locations in memory, allowing you to treat each element of the array differently, but only requiring one variable to store all the data.

Initializing and Accessing

Below is the declaration, initialization, and accessing of an array. 

ArrayType[] arrayName; // Declaration
arrayName = new ArrayType[arrayLength]; // Initialization
arrayName[index] // Accessing the index-th element.

To make it convenient for the programmer, if you know all the elements you want to add to an array, you can initialize an array with an array literal. Below is an example. 

String[] names = {"Brian", "Johnny", "Isaac", "Jeffrey", "Jonathan", "Daniel"};

The default values of each element of an array will be the default value of that data type. That is, null for objects, 0 for ints, 0.0 for doubles, etc. To access any specific element, you can access it by putting the desired index in square brackets. It is important to remember that arrays are zero-indexed. This means that the first element is index 0, second index 1, etc, where the last one is the length of the array - 1. You can access the length of the array with the public instance variable length.

When creating an array, the Java compiler will only accept a 32-bit integer as the parameter, so if you would not be able to create an array with a length of 8 billion, one for each person on Earth.

Potential Errors

When you create or access an array with a data type that isn't an int, a TypeMismatchException will be thrown. However, what if you input a negative integer for array size or an index that isn't valid?

NegativeArraySizeException

This is an exception specifically thrown when trying to initialize an array size to a negative value.

ArrayIndexOutOfBoundsException

This is an exception thrown when trying to access an index of an array that is out of bounds, either greater than or equal to the size of the array, or a negative integer.

Multidimensional Arrays

Multidimensional arrays, to put simply, are arrays with more than one dimension. For instance, a 2D array is an array of arrays.

They are great when used to store many different aspects of data related to a singular group of people. For instance, we can have a 2D array to store the age, grade, ICS grade, number of absences, birth date, and birth month of a class of students. One dimension of the array would represent each student, while the other would represent each property of the student.

Example

For this unit, our assignment was to use parallel arrays to represent the data regarding the achievements of students running for Student Council. We would then print out the winner, along with their score calculated with our custom formula. Our program worked by storing data of the same person in the same index across all arrays, so it would sort of act as a 2D array, the same person index is used throughout all the sets of data. Below are some code snippets of the implementation.

Declaration and Initialization

private int PEOPLE; //Length
private String[] name;
private double[] gpa;
private int[] volunteerHours;
private String[] previousStudentCouncilPosition;
private int[] grade;

PEOPLE = 5;
name = new String[PEOPLE];
gpa = new double[PEOPLE];
volunteerHours = new int[PEOPLE];
previousStudentCouncilPosition = new String[PEOPLE];
grade = new int[PEOPLE];

Accessing

for(int i = 0;i < PEOPLE;i++){
    if(max < grade[i]){
        max = grade[i];
        bestPeople = "";
        bestPeople+=i + "";
    }else if(max == grade[i]){
        bestPeople+=i + "";
    }
}

Output

Name: Candidate B
Grade: 11
GPA: 96.80
Volunteer Hours: 199
Previous Position: N/A
